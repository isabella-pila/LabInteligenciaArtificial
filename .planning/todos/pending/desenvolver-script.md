# Desenvolver e executar script do Perceptron

Desenvolver o script principal do Perceptron contendo a lógica de treinamento pela Regra de Hebb.
Rodar as 5 execuções (conforme REQ-002) e registrar os pesos finais e o número de épocas/erros de cada uma.
